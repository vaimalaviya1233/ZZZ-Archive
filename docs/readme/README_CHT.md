## 跨平台絕區零維基應用程式

ZZZ Archive 是動作遊戲 "絕區零" 的維基應用程式，使用者可查詢遊戲中相關資料、瀏覽熱門的二創作品。

## 桌面端執行方法

- Desktop App

```
./gradlew run
```

- Desktop Hot Reload

```
./gradlew runHot
```
